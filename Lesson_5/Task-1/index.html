Инструкция:

  • Рисовать можно двумя цветами (Color 1 и Color 2)
  • Для выбора цвета Color 1 кликните в нужный цвет на палитре
  • Для выбора цвета Color 2 кликните в нужный цвет на палитре удерживая клавишу command(⌘) (или ctrl для windows/linux)
  • Для рисования цветом Color 1 кликните по холсту
  • Для рисования цветом Color 2 кликните по холсту удерживая клавишу command(⌘) (или ctrl для windows/linux)
  • Что бы стереть точку на холсте кликните по точке удерживая клавишу shift(⇧)
